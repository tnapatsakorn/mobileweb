❓ ถาม-ตอบ
1. ทำไมจึงควรใช้ Structured Output แทนการ parse text ปกติ?
Structured Output กำหนด JSON Schema ไว้ล่วงหน้า ทำให้ได้ข้อมูลที่มีโครงสร้างแน่นอนทุกครั้ง ไม่ต้องเขียน regex หรือ parse text เอง ลดโอกาส error และง่ายต่อการนำไปแสดงผลใน UI
2. หากไม่ใช้ Firebase AI Logic แล้วเรียก Gemini API ตรงๆ จะมีความเสี่ยงอะไร?
API Key จะถูกฝังอยู่ใน client-side code ทำให้ผู้ไม่หวังดีสามารถดึง key ไปใช้งานได้ อาจทำให้เกิดค่าใช้จ่ายสูง หรือถูก abuse ได้ Firebase AI Logic ช่วยซ่อน key ไว้ฝั่ง server และรองรับ App Check เพื่อป้องกัน
3. base64 มีข้อเสียอย่างไรเมื่อไฟล์ภาพมีขนาดใหญ่?
base64 จะเพิ่มขนาดข้อมูลประมาณ 33% เมื่อเทียบกับไฟล์ต้นฉบับ ทำให้ใช้ memory มากขึ้น ส่งผลให้ request payload ใหญ่ขึ้น ใช้เวลา upload นานขึ้น และอาจทำให้แอปช้าหรือ crash บนมือถือที่ RAM น้อย
