❓ ถาม-ตอบ
นักศึกษาใช้ AI ช่วยส่วนไหนบ้าง?
ใช้ AI ช่วยสร้างโครงสร้างโปรเจกต์ทั้งหมด ได้แก่ Core Module (types.ts, MotionService, TtsService, HapticsService, ArmWorkoutEngine) และ UI ใน HomePage.vue รวมถึงการออกแบบ CSS Dark Theme และช่วยแก้ TypeScript error ต่างๆ
ส่วนใดบ้างที่นักศึกษาปรับแก้ไขเอง?
